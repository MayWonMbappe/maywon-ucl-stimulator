# UCL Simulator Deep Fix 2026-06-20

本版修复：

1. 联赛阶段赛后轮次推进：踢完第 1 轮立即进入第 2 轮赛前状态，并要求完成轮间管理；第 8 轮后才结算联赛阶段。
2. 瑞士轮对手层次：8 轮对手按四个强度层各抽两队，并按中等-弱-强-中等-中等-弱-强-中等交错，避免前强后弱。
3. 对手调整：新增多种对手模糊调整文本，并且会改变后续 xG/反击/压迫/定位球/纪律风险。
4. 事件结果叙述：不再使用单一“机会形成但未改写比分”，改为球员动作 + 具体结果，如偏出、中柱、被扑、横梁、拦截、黄牌、两黄变红。
5. 红黄牌与停赛：两黄变红后该球员本场后续事件不再出现；下场自动停赛，阵型图自动补入可用替补；再下一场恢复。
6. 赛季结束雷达图：新增 8 维百分制雷达图、A+ 至 F 评级；指标点附近透明热区可显示分数、评级和要求。
7. 保留 public/ + Worker 兜底部署结构，避免根路径 404。
